== Description ==

[ElementorKit](https://elementorkit.site) is a powerful WordPress plugin that enables you to access thousands of premium Elementor template kits and millions of royalty-free stock photos without leaving WordPress.

**⚠️ Note:** ElementorKit is the replacement for the discontinued [Envato Elements WordPress plugin](https://help.elements.envato.com/hc/en-us/articles/49783425668761-Sunsetting-Extensions).

**Key Features:**
* Browse and import Elementor template kits directly inside WordPress
* Access over 12 million royalty-free stock photos
* Manual upload and import of template kit files supported
* Fully compatible with Elementor and Elementor Pro
* Works with any WordPress theme
* Premium features available for ElementorKit subscribers

== Installation ==

1. Upload the `elementorkit` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Connect your ElementorKit API key to unlock premium template kits and stock photos
4. Start browsing and importing template kits and photos directly inside WordPress

== Frequently Asked Questions ==

= Do I need an ElementorKit subscription? =

You can start with free template kits. To access premium template kits and stock photos, an ElementorKit subscription is required.

= What is a Template Kit? =

A Template Kit is a collection of templates designed for a specific niche, ready to import into Elementor for professional website creation.

= How do I access Premium Templates and Photos? =

ElementorKit subscribers can connect their API key to unlock hundreds of premium Template Kits and millions of stock photos for direct import into WordPress.

= Will Template Kits work with my theme? =

Yes! Templates are compatible with any WordPress theme.

= Will it work with my favourite page builder plugin? =

Currently, Elementor and Elementor Pro are supported.

= Does the plugin use an external service? =

Yes. ElementorKit uses a secure service hosted on [elementorkit.site](https://elementorkit.site) to provide template kits, stock photos, and related content. Only plugin version and requested templates/photos are recorded.

= How do I get support or provide feedback? =

For questions, issues, or feedback, visit [https://elementorkit.site/contact-us/](https://elementorkit.site/contact-us/).
